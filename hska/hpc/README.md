#### Speedup

| Variable | Bedeutung |
| ----- | ----- |
| $Ts$ | Sequentielle Laufzeit |
| $Tp$ | Parallele Laufzeit |
| $S$  | Speedup |

$S = \frac{Ts}{Tp}$


#### Effizienz

| Variable | Bedeutung |
| ----- | ----- |
|$E$ | Effizienz |
|$P$ | Anzahl Prozessoren (optimaler Speedup) |

$E = \frac{S}{P}$
